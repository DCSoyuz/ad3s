# AD3S — программное обеспечение для микросхемы 5400ТР065А-022

Открытый комплект ПО для настройки, отладки и применения микросхемы преобразователя угол→код **5400ТР065А-022** (АО «Дизайн Центр «Союз»). Репозиторий содержит два компонента, которые работают в паре: **настольное приложение** (ПК) и **прошивку моста** на ESP32-S3 между ПК и микросхемой.

```
┌─────────────────────┐   USB CDC    ┌───────────────┐   SPI   ┌─────────────────────┐
│ workstation4ad3s    │ ◄──────────► │  ESP32-S3     │ ◄─────► │  5400ТР065А-022     │
│ (Java, FlatLaf)     │  виртуальный │  ad3s_prog_   │  10 МГц │  (RDC, ДЦ «Союз»)   │
│                     │   COM-порт   │  esp32        │         │                     │
└─────────────────────┘              └───────────────┘         └─────────────────────┘
```

## Состав репозитория

| Каталог | Что это | Технологии |
|---------|---------|------------|
| [`workstation4ad3s/`](workstation4ad3s/README.md) | Настольное приложение для настройки и отладки микросхемы | Java 17+, Swing + FlatLaf, Maven |
| [`ad3s_prog_esp32/`](ad3s_prog_esp32/README.md) | Прошивка моста ESP32-S3 (USB ↔ SPI) | ESP-IDF (C/C++) |

Подробности сборки, настройки и использования — в README каждого каталога.

## С чего начать

**Вариант 1 — готовое приложение (без сборки).** Скачайте JAR и скрипты запуска с FTP: <https://ftp.dcsoyuz.ru/soft/workstation4ad3s/> (требуется Java 17+). Прошивку моста (исходники) — на GitHub: <https://github.com/DCsoyuz/ad3s/tree/main/ad3s_prog_esp32>.

**Вариант 2 — собрать из исходников.** См. README соответствующего каталога: [`workstation4ad3s/`](workstation4ad3s/README.md) и [`ad3s_prog_esp32/`](ad3s_prog_esp32/README.md).
